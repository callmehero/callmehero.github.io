一、什么是hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
详细：https://hexo.io/zh-cn/docs/index.html

四、环境的配置

GIT的配置

1：安装好git之后，再开始菜单中打开 “Git Bash”,如下配置
email和name分别对应上github上面的。

2添加SSH密钥到GitHub, 检查本机现有的SSH密钥
打开你的Git Bash， 输入
$ ls -al ~/.ssh

3创建你的SSH
$ ssh-keygen -t rsa -C "注册Github用的邮箱"
注意期间“输入密码（空没有密码）：再次输入密码]：[键入密码]＃再次输入相同的密码”，
如下：
“Enter passphrase (empty for no passphrase): [Type a passphrase] # Enter same passphrase again:”
可不管，直接一路ENTER。

4保存你的SSH并将其添加到你的github
$ clip < ~/.ssh/id_rsa.pub
点击link，进入你github的setting，如下图

把刚获取的SSH填入KEY，title 随便取个就好啦。

hexo 的配置

安装Node.js后，所有必备的应用程序安装完成后，即可使用 npm(Node.js自带无需另行安装) 安装 Hexo。
==运行cmd==
安装hexo之前初始化来创建package.json前提配置一定要做好不然很容易踩坑哦

$ npm init 初始化操作

$ npm install -g hexo-cli 改命令用于安装hexo
